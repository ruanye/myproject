# 作业项目接口文档

## 用户模块

域名：http://132.232.89.22:8083

1.  用户注册
    接口地址 : /register
    请求方式：post
    参数:
    |参数名| 类型 |是否必须|
    |---|---|----|
    |username |String | 是|
    |password|String | 是|
    ***
    返回值：
    | 值 | 类型 | 说明 |
    | ---- | ------ | ------------------ |
    | code | int | 200 成功，400 失败 |
    | msg | string | 请求失败原因 |

请求示例子

```js
axios.post('/register', {
  username: 'lili',
  password: '123456'
});
```
